Instalar Pacotes:

  • npm install lodash -> instale no arquivo o lodash
  • npm install lodash eslint -g -> Instale o lodash globalmente
  • npm update lodash -> atualize a versão mais recente do lodash
  • npm uninstall lodash -> Desinstalar o Lodash

Package.Json

  • Package.Json -> Arquivo local com as configurações e dependências de pacotes NPM.
  • npm init -> Inicia uma nova configuração local do NPM
  • npm install -> Instala todas as dependêncais listadas no arquivo package.Json
